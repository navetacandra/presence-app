## PIN

| ESP     | OUT         |
|---------|-------------|
| D1      | LCD I2C SCL |
| D2      | LCD I2C SDA |
| D3      | RFID RST    |
| D8      | RFID SDA    |
| D5      | RFID SCK    |
| D6      | RFID MISO   |
| D7      | RFID MOSI   |
| D4      | BUZZER      |
